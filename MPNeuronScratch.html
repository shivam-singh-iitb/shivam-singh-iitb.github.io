MP Neuron Model¶

The McCulloch Pitts Model or the MP Neuron Model is the simplest of the Neural Networks built.
The six jars of the model are :

  • Model : y_pred = ( $\sum_{i=1}^n$ x_i >= $\theta$ ) where $\theta$ = Threshold for the model
  • Data : The input to the MP Neuron model is binary and the output too is binary,i.e. X $\in$ {0,1} and y $\in$ {0,1}
  • Task : Predict a Boolean output,i.e. Y_pred $\in$ {0,1} where 0 : the output is Negative and 1 : output is Positive.
  • Loss Function : The loss function used is the squared error loss : $\sum_{i=1}^n (y\_pred - y)^2$. We could as well use the absolute error loss : $\sum_{i=1}^n |y\_pred - y|$ but absolute value function is not differentiable, hence, squared error loss is used.
  • Learning Algorithm : The only parameter to learn is the threshold value $\theta$. The value can be learnt in two ways:
    • Brute Force Method : $\theta$ can have discrete values from 0 upto n, as $\sum_{i=1}^n$ x_i can take up value 0 when all x_i's are 0 and n when all xi's are 1. So, we can check the loss function for all values of $\theta$ in range(0,n) and select the optimum $\theta{opt}$ to be the one with minimum loss.
    • Optimized Method : Not discussed in the following code but can be implemented easily.
      1. Choose $\theta$ = int(n/2) and in each epoch do,
      2. Predict the output using the current value, and find Confusion Matrix
      3. if false positive < false negative and $\theta$ < n: do $\theta$ += 1
        else if false positive > false negative and $\theta$ >= 1 : do $\theta$ -= 1
        else $\theta$ remains unchanged`
  • Evaluation : The evaluation metric is the accuracy score=$\frac{Number of Correct Predictions}{Total Number ofPredictions}$

class mpNeuron:
    def __init__(self,theta=0):
        self.theta = theta   
    
    def predict(self,X):
        y_pred = [1 if sum(x)>=self.theta else 0 for x in np.asarray(X)]
        return y_pred
    
    def fit(self,X,y,):
        theta_values = [i for i in range(X.shape[1]+1)]
        accuracy = []
        loss = []
        for theta in theta_values:
            self.theta = theta
            y_pred = self.predict(X)
            accuracy.append(accuracy_score(y,y_pred))
            loss.append(mean_squared_error(y,y_pred)*X.shape[1])
            
        plt.figure(figsize=(20,8)) 
        plt.subplot(1,2,1)
        plt.scatter(theta_values,accuracy)
        opt_theta = accuracy.index(max(accuracy))
        max_accuracy = max(accuracy)
        plt.plot(opt_theta,max_accuracy,"bo")
        annot = "Best Fit\n"+"Theta :" + str(opt_theta) + "\nAccuracy :" \
        + str(round(max_accuracy,2))
        plt.annotate(annot,xy=(opt_theta,max_accuracy),xytext=(0,0.6),
                     arrowprops=dict(facecolor="gray",shrink=0.05))
        plt.title("Accuracy vs Threshold")
        plt.xlabel("Threshold value")
        plt.ylabel("Accuracy Score")
        self.theta = opt_theta
        plt.subplot(1,2,2)
        min_loss = loss[theta_values.index(opt_theta)]
        plt.scatter(theta_values,loss)
        plt.plot(opt_theta,min_loss,"bo")
        annot = "Best Fit\n"+"Theta :" + str(opt_theta) + "\nLoss :" \
        + str(min_loss)
        plt.annotate(annot,xy=(opt_theta,min_loss),xytext=(0,15),
                     arrowprops=dict(facecolor="gray",shrink=0.05))
        plt.title("Loss vs Threshold")
        plt.xlabel("Threshold value")
        plt.ylabel("Squared Error Loss")
